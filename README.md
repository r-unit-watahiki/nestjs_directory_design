# nestjs_directory_design

## プロジェクト作成

```bash
npx @nestjs/cli new . --package-manager npm --strict
```

## サーバー起動

```bash
# 開発モード（ホットリロード付き）
npm run start:dev

# 通常起動
npm run start

# 本番モード
npm run start:prod
```

## テスト実行

```bash
# ユニットテスト
npm run test

# E2Eテスト
npm run test:e2e

# カバレッジ
npm run test:cov
```
